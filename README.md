My Image Gallery Repository

このリポジトリは、画像を保管し、GitHub Pagesを使用してWeb上で一覧表示するためのものです。

フォルダ構成

images/: 全ての画像ファイルをこのフォルダ内に保存してください。

index.html: アップロードした画像を自動的に表示するメインページです。

画像の追加方法

images/ フォルダ内に画像をアップロードします。

index.html 内の imageData 配列に、新しい画像のパスと情報を追記してください。

GitHubにコミットすると、自動的にWebページが更新されます。

Webサイトの閲覧方法

GitHubの Settings > Pages から公開設定を有効にしてください。
URL: https://<あなたのユーザー名>.github.io/<リポジトリ名>/
